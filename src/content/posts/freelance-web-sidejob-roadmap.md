## はじめに

「Web制作の副業で月10万円」は、正しいステップを踏めば十分に達成可能な目標です。

ただし、闇雲にスキルを学んでも案件は取れません。この記事では、未経験からWeb制作の副業を始めて月10万円を達成するまでの具体的なロードマップを解説します。

## フェーズ1: スキル習得（1〜2ヶ月）

### 最低限必要なスキル

副業Web制作で案件を受注するために、最低限以下のスキルが必要です。

- **HTML/CSS**: レイアウト、レスポンシブ対応
- **JavaScript**: DOM操作、フォームバリデーション
- **デザインツール**: Figma の基本操作（デザインカンプの読み取り）
- **WordPress** または **静的サイトジェネレーター**: 納品形式として必須

### 学習の優先順位

すべてを完璧にする必要はありません。以下の順番で学ぶのが効率的です。

1. HTML/CSS の基礎（2週間）
2. レスポンシブデザイン（1週間）
3. JavaScript の基礎（1週間）
4. WordPress または Astro でのサイト構築（2週間）

### 避けるべき学習の罠

- **フレームワークの深追い**: React や Vue は副業LP制作にはオーバースペック
- **完璧主義**: 80%理解したら次に進む。残りは実案件で学ぶ
- **インプット過多**: チュートリアルは3つまで。あとは手を動かす

## フェーズ2: ポートフォリオ作成（2週間）

案件に応募するには、実績を見せるポートフォリオが必要です。

### ポートフォリオに載せるもの

| 項目 | 内容 | 目安 |
|------|------|------|
| 架空LP | 飲食店・美容室などの LP | 3〜5本 |
| 自己紹介 | スキル・対応可能な作業 | 1ページ |
| 制作実績 | スクリーンショット + 説明 | 各作品ごと |

### ポートフォリオのポイント

- **実際のビジネスを想定**: 「架空の美容室」など具体的な設定で作る
- **レスポンシブ対応必須**: クライアントはスマホで確認する
- **表示速度を意識**: Lighthouse スコア90以上を目指す
- **コードの品質**: GitHub にソースを公開して技術力をアピール

## フェーズ3: 案件獲得（1ヶ月〜）

### 案件を探す場所

副業Web制作の案件は主に以下のプラットフォームで見つかります。

- **CrowdWorks**: 案件数が最も多い。初心者向けも豊富
- **Lancers**: CrowdWorks と並ぶ大手。提案形式の案件が多い
- **ココナラ**: 自分でサービスを出品する形式。単価設定の自由度が高い

### 応募文のコツ

クライアントは大量の応募文を読んでいます。以下を意識すると通過率が上がります。より詳しい応募文の書き方は[クラウドソーシングで選ばれる応募文の書き方](/blog/crowdsourcing-proposal-writing-tips/)で解説しています。

1. **冒頭3行で結論を書く**: 何ができるか、いつまでにできるか
2. **相手の課題に触れる**: 案件説明を読み込み、具体的な提案をする
3. **ポートフォリオを添付**: 類似案件の実績があれば最優先で見せる
4. **テンプレ感を出さない**: 案件ごとにカスタマイズする

### 初案件の単価目安

| 案件タイプ | 相場 | 作業時間目安 |
|-----------|------|------------|
| LP 1ページ | 3〜5万円 | 15〜25時間 |
| WordPress サイト（5P） | 5〜10万円 | 30〜50時間 |
| コーディングのみ | 2〜3万円 | 10〜15時間 |

最初は相場より少し低めで受注し、実績を積むのが定石です。適正な単価の考え方は[副業Web制作の見積もり・価格設定ガイド](/blog/freelance-pricing-guide/)で詳しく解説しています。

## フェーズ4: 単価アップ（3ヶ月〜）

月10万円を安定させるには、単価を上げるか効率を上げるかの2択です。

### 単価を上げる方法

- **対応範囲を広げる**: デザイン + コーディングのセットで受注
- **専門性を持つ**: 特定業界（医療・不動産など）に特化する
- **直接営業**: プラットフォームの手数料を省ける
- **継続案件を確保**: 保守・運用契約で安定収入を作る

### 効率を上げる方法

- **テンプレート化**: よく使うセクションをコンポーネント化
- **AI ツール活用**: [Claude Code](/blog/claude-code-web-development/) などでコーディング速度を向上
- **作業の定型化**: ヒアリングシート、納品チェックリストを用意

## 月10万円の内訳シミュレーション

現実的なシミュレーションを見てみましょう。

```
LP制作 × 2件 = 8万円（1件4万円）
WordPress 保守 × 2件 = 2万円（月1万円/件）
─────────────────────
合計: 10万円/月
```

LP制作を月2件こなし、保守案件を2件持つのが最も再現性の高いパターンです。

## よくある失敗パターン

### 1. 安すぎる単価で消耗する

1件5,000円の案件を20件こなすのは現実的ではありません。最低でも1万円以上の案件に応募しましょう。

### 2. スキル習得に時間をかけすぎる

完璧を目指すと永遠に案件に応募できません。最低限のスキルで応募を始め、実案件で成長するのが最速です。

### 3. 1つのプラットフォームに依存する

CrowdWorks だけ、Lancers だけに依存すると案件の波に影響されます。複数のチャネルを持つことが安定の鍵です。

## まとめ

副業Web制作で月10万円を達成するまでのステップは以下の通りです。

1. **スキル習得**（1〜2ヶ月）: HTML/CSS、JS、WordPress/Astro の基礎
2. **ポートフォリオ作成**（2週間）: 架空案件3〜5本を制作
3. **案件獲得**（1ヶ月〜）: CrowdWorks/Lancers で応募開始
4. **単価アップ**（3ヶ月〜）: 対応範囲拡大と効率化

最初の1件を受注するまでが最も大変ですが、実績が積み上がれば案件獲得は加速度的に楽になります。まずは行動を始めることが最も重要です。
