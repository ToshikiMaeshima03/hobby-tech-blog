## はじめに

クラウドソーシングで案件を獲得できるかどうかは、**応募文の質**でほぼ決まります。

CrowdWorks や Lancers の人気案件には数十件の応募が集まります。クライアントは大量の応募文を短時間で判断するため、冒頭数行で「この人に頼みたい」と思わせなければ、どれだけスキルがあっても選ばれません。

この記事では、受注率を上げる応募文の書き方を、テンプレートとNG例付きで解説します。

## クライアントが応募文で見ているポイント

クライアント側の視点を理解することが、良い応募文を書く第一歩です。

| 優先度 | チェックポイント | 理由 |
|-------|--------------|------|
| 高 | 案件内容を理解しているか | テンプレ応募の排除 |
| 高 | 具体的な実績があるか | スキルの証明 |
| 中 | 納期・進め方が明確か | プロジェクト管理能力 |
| 中 | 料金が妥当か | 予算との適合性 |
| 低 | 自己紹介の長さ | 簡潔さ＝読みやすさ |

最も重要なのは「この案件を理解している」ことが伝わるかどうかです。

## 選ばれない応募文のNG例

まず、典型的なNG応募文を見てみましょう。

```
はじめまして。Web制作を行っている○○と申します。
HTML/CSS/JavaScript/PHP/WordPress に対応可能です。
レスポンシブ対応もできます。
ぜひお任せください。よろしくお願いいたします。
```

この応募文の問題点は以下の通りです。

- **案件への言及がゼロ**: どの案件にも送れるテンプレ感が強い
- **具体性がない**: 「対応可能」だけではスキルレベルがわからない
- **実績の提示がない**: クライアントが判断する材料がない
- **差別化がない**: 他の応募者との違いが見えない

## 選ばれる応募文の構成

効果的な応募文は以下の5パートで構成します。

### パート1: 案件への理解を示す（冒頭3行）

```
はじめまして。Web制作を行っている○○です。

「飲食店のLP制作で、予約数を増やしたい」というご要望を拝見しました。
飲食業界のLP制作経験があり、お力になれると思いご応募いたします。
```

冒頭で**案件の具体的な内容に触れる**ことで、「ちゃんと読んでいる」ことが伝わります。

### パート2: 関連実績を提示する

```
■ 関連実績
・飲食店LP制作（イタリアンレストラン）: https://example.com/demo1
  → 公開後、Web予約数が月30件→65件に増加
・カフェのコーポレートサイト制作: https://example.com/demo2
  → Lighthouse パフォーマンス 98点

上記の経験から、飲食業界に特化した訴求ポイントの配置や
予約導線の設計について知見があります。
```

数字を含めた実績を提示すると説得力が格段に上がります。ポートフォリオのURLは必ず添付しましょう。なお、見積もりの出し方に不安がある方は[副業Web制作の見積もり・価格設定ガイド](/blog/freelance-pricing-guide/)も参考にしてください。

### パート3: 具体的な提案をする

```
■ ご提案
今回のLP制作にあたり、以下の構成を想定しています。

1. ファーストビュー（店舗の魅力が伝わる画像 + 予約CTA）
2. メニュー紹介（人気メニュー3品をビジュアル訴求）
3. 店内の雰囲気（写真ギャラリー）
4. アクセス・営業時間
5. 予約フォーム（即座に予約完了するフロー）

特に「予約フォームまでの導線」を最短にすることで、
離脱を防ぐ設計を重視します。
```

具体的な提案を含めることで、「この人は仕事の進め方がイメージできている」と感じてもらえます。

### パート4: 納期・進め方

```
■ スケジュール
・ヒアリング: 受注後2日以内
・デザイン案提出: 5日後
・コーディング完了: 10日後
・修正対応: 2回まで（計14日で納品）

進捗は Chatwork/Slack/メール いずれでもご報告可能です。
```

### パート5: 料金と締め

```
■ 料金
ご予算の○万円で対応可能です。
修正2回分を含んだ価格です。

ご不明点やご質問がありましたら、お気軽にお申し付けください。
ご検討のほど、よろしくお願いいたします。
```

## 応募文テンプレート（コピペ用）

上記を踏まえた汎用テンプレートです。【 】内を案件に合わせてカスタマイズしてください。

```
はじめまして。Web制作を行っている【名前】です。

【案件の要約を1〜2行で記載】を拝見し、
【関連する経験・スキル】の経験からお力になれると思いご応募いたします。

■ 関連実績
・【実績1のタイトル】: 【URL】
  → 【成果を数字で記載】
・【実績2のタイトル】: 【URL】
  → 【成果を数字で記載】

■ ご提案
【案件に対する具体的な提案を3〜5行で記載】

■ スケジュール
・ヒアリング: 受注後【X】日以内
・初稿提出: 【X】日後
・納品: 【X】日後（修正【X】回含む）

■ 料金
ご予算の【X万円】で対応可能です。

ご不明点がありましたら、お気軽にお申し付けください。
ご検討のほど、よろしくお願いいたします。
```

## 受注率を上げる7つのテクニック

1. **応募はスピードが命**: 公開直後の案件に応募すると競合が少なく有利
2. **1案件1応募文**: コピペを使い回さず、毎回カスタマイズする
3. **プロフィールを充実させる**: 応募文と合わせてプロフィールも見られる
4. **料金は範囲で提示**: 「5〜8万円」のように幅を持たせると柔軟に見える
5. **質問を1つ入れる**: 「ご希望のデザインの方向性はありますか？」でやり取りのきっかけを作る
6. **添付ファイルを活用**: ポートフォリオPDFやスキルシートを添付する
7. **応募後にフォロー**: 数日返信がなければ丁寧にフォローメッセージを送る

## カテゴリ別の応募文ポイント

| 案件カテゴリ | 応募文で強調すべきポイント |
|------------|---------------------|
| LP制作 | CVR改善の実績、マーケティング視点 |
| コーディング | 表示速度（[Core Web Vitals](/blog/core-web-vitals-improvement-guide/)の知識）、コードの品質、ピクセルパーフェクト |
| WordPress | セキュリティ対策、保守・運用の提案 |
| デザイン | ポートフォリオの見せ方、ブランディング理解 |
| SEO対策 | 具体的な数値改善実績、使用ツール |

## まとめ

選ばれる応募文のポイントを振り返ります。

- **冒頭3行**で案件への理解を示す
- **関連実績**を数字付きで提示する
- **具体的な提案**で仕事のイメージを伝える
- **納期・料金**を明確にしてクライアントの判断を助ける
- **テンプレを使い回さない**: 案件ごとに必ずカスタマイズする

応募文は「自分を売り込む営業資料」です。時間をかけて丁寧に書くことが、受注率の向上に直結します。副業Web制作を始めたばかりの方は[副業Web制作で月10万円を達成するロードマップ](/blog/freelance-web-sidejob-roadmap/)で全体像を把握するのもおすすめです。
